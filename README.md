# MCBEOPItemGiveAdd-on / MCBEOIG 

ScriptAPIを使ったUI型で使いやすいOPアイテム入手アドオンです。

![Image](https://github.com/user-attachments/assets/daecaeb7-7729-4bfd-9127-189c9e186002)

[ダウンロードはこちらから](https://github.com/NaruCommandLaboratory/MCBEOPItemGiveAdd-on/releases/tag/v1.0.0)

# 使い方 

ワールド生成時に、実験的機能の"ベータAPI"と"EducationEdition"を必ず有効にしてください。 （有効にしないとアドオンが動きません） 

ワールドが生成されたら、チャット欄で"!opitemgive"と入力して実行すると、OPアイテムを入手できるUIが表示されます。 

# こんなときはQ＆A 

Q, アドオン入れたのに何も反応しません。 

A. 上の 使い方 の手順に従ってください。ワールドの実験的機能を有効にし忘れてることが多いです。

Q, Realmsサーバーでうまく機能しない 

A, ScriptAPIはRealmsや一部のマルチプレイ環境では動かない場合があります。ご了承ください。 

Q, アイテムが付与されない 

A, アドオンを抜いてもう一度入れてみてください（それでも付与されない場合は、[Discord](https://discord.gg/98vzZsArdc)でお知らせください）
